# 2021-22-Class-Cal-Generator

The Haverford School 2021-22 School Year

This program generates a CSV file with all your classes. This spreadhseet can be imported into your calendar. **It is HIGHLY recommended to create a new calendar to import the CSV file into. If you do not and there is a mistake with your classes, you will have to manually dleete 1000+ calendar events.** There are links below on hown tocreate a new calendar and import the file.
